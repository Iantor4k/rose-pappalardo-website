# Website update quick-start

This folder is connected to your GitHub website repository.

## How to use it

1. Open this folder in Cursor.
2. In chat, describe the website change you want in plain English.
3. Ask: "Please make the change and push it to GitHub."
4. I will edit files, commit, and push for you.

## Good examples to type

- "Change the contact email on the contact page and push."
- "Add a new painting to the gallery and push."
- "Update the about text and publish."

## Notes

- You are currently authenticated to GitHub, so push should work.
- If GitHub login expires in the future, I can help reconnect in a minute.
